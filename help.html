Help Center

Need help?

Welcome to the help center. Here are the most common tasks you can do in SmartExam.

How do I create a new test?

Go to the Create Test section, enter the test details, then add questions using the "Add question" button. When you're done, click Save test.

Where can I manage assignments?

Use the Assignments section to publish assignments for students. They're stored automatically and will appear for students under the Assignments page.

How do I view student results?

In the Analytics section, you can review stored test results and export them as a CSV file.

Can I edit questions after saving?

Yes — go to the Manage Question Bank page to edit or delete questions that can be reused across tests.

Where are files stored?

All data is stored in the browser localStorage for now. This means it stays on your machine unless you clear browser data.
